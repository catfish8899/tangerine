tangerine 桌面 agent框架 - 开发说明书
一、技术栈与核心依赖
桌面壳层：Tauri 2（Rust）
前端：React 18 + TypeScript + Vite 6 + Tailwind CSS v4
工作流画布：@xyflow/react
Markdown 渲染：
react-markdown
remark-gfm
rehype-highlight
highlight.js
Tauri 插件：
@tauri-apps/plugin-shell
@tauri-apps/plugin-dialog
@tauri-apps/plugin-fs
Python Sidecar：FastAPI
Rust 图片处理：base64
二、项目目录结构
Text
│   │   ├── settings/                   # 设置面板解耦子模块
│   │   │   ├── types.ts                # 设置面板相关的类型定义 (ApiProviderConfig 等)
│   │   │   ├── constants.ts            # 设置面板相关的常量、Storage Key 与默认配置
│   │   │   ├── useSettingsManager.ts   # 设置面板核心状态管理与业务逻辑 Hook
│   │   │   ├── SettingsSidebar.tsx     # 设置面板左侧导航栏
│   │   │   ├── SettingsHeader.tsx      # 设置面板顶部标题与关闭按钮
│   │   │   ├── ApiProviderCard.tsx     # 单个 API 提供商配置卡片（复用于云端和 Ollama）
│   │   │   ├── CloudModelsTab.tsx      # 云端模型设置 Tab 内容
│   │   │   ├── OllamaModelsTab.tsx     # Ollama 模型设置 Tab 内容
│   │   │   └── FormatSettingsTab.tsx   # 对话格式设置 Tab 内容
│   │   └── automation/
│   │       ├── automationConstants.ts  # 画布常量
│   │       ├── automationTypes.ts      # 画布类型定义
│   │       ├── automationStorage.ts    # 画布存储封装
│   │       ├── automationNodeUtils.ts  # 纯函数工具：ID、快照、集合关系、子孙查找等
│   │       ├── AutomationNodes.tsx     # 自定义节点组件与 nodeTypes
│   │       ├── AutomationToolbar.tsx   # 底部工具栏；Pointer 拖拽起点
│   │       ├── DragGhost.tsx           # 拖拽幽灵预览
│   │       ├── SystemTimeBadge.tsx     # 系统时间徽标
│   │       └── FlowEditor.tsx          # 主画布编辑器；节点、边、快捷键、复制粘贴、撤销重做
│   ├── hooks/
│   │   └── useChatManager.ts           # 聊天核心状态管理；会话、流式响应(AbortController中断)、分支切换与保存、深度删除防复活、附件与拖放
│   ├── types/
│   │   └── chat.ts                     # 聊天域类型定义；含 Message(含isStopped/branches) / ChatSession / Role / AttachmentFile
│   └── App.tsx                         # 应用主入口；聊天页/自动化页主视图分流、全局右键菜单(调用深度删除/重发/编辑)
└── src-tauri/
    ├── capabilities/
    │   └── default.json                # Tauri capability 配置
    ├── src/
    │   └── lib.rs                      # Rust 主入口；插件注册、Sidecar 拉起、read_image_as_data_url 命令
    ├── Cargo.toml                      # Rust 依赖清单
    ├── tauri.conf.json                 # Tauri 配置
    └── py_sidecar/
        ├── main.py                     # FastAPI 服务；/chat SSE流式输出、客户端断开检测(Request.is_disconnected)以切断算力、多模型适配
        ├── debugger.py                 # 调试面板
        └── requirements.txt            # Python 依赖
