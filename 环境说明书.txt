# Tangerine 开发环境一键配置指南

## 1. 跨电脑克隆后的全局前置要求
在目标电脑上，请确保已手动安装以下基础环境：
- **Node.js**: v18.x 或 v20.x LTS
- **pnpm**: 全局包管理器
- **Rust 环境**: 安装 `rustup`，Windows 电脑确保已勾选“VS C++ 桌面开发”工作负载。
- **Python 3**: 确保 `python` 已加入系统环境变量 PATH。

## 2. 必须配置的系统环境变量 (API Keys)
请在 Windows 系统环境变量中配置以下密钥（或写入你的本地 `.env` 环境）：
- `DEEPSEEK_API_KEY` - 云端 DeepSeek 推理秘钥
- `GEMINI_API_KEY` - 云端 Gemini 推理密钥
- `TAVILY_API_KEY` - Tavily 联网搜索 API 密钥

---

## 3. 快速启动命令（新电脑一键复制执行）

### 步骤 A：在根目录下执行，安装前端及画布依赖
```bash
pnpm install && pnpm tauri add shell && pnpm add @xyflow/react react-markdown remark-gfm rehype-highlight highlight.js
步骤 B：安装 Python Sidecar 依赖
Bash
pip install -r src-tauri/py_sidecar/requirements.txt
步骤 C：(可选) 清理本地旧编译缓存（针对 PowerShell）
PowerShell
if (Test-Path .vite) { Remove-Item -Recurse -Force .vite }
步骤 D：一键拉起 Tauri 桌面客户端预览
Bash
pnpm tauri dev
步骤 E：(可选) 另开终端启动后台硬件级时序调试面板
Bash
cd src-tauri/py_sidecar/ && python debugger.py
